To set up the SDDM astronaut theme for your wallpaper manager, follow these steps:

1. Install the theme from the repository:

[SDDM Astronaut Theme](https://github.com/Keyitdev/sddm-astronaut-theme)

2. Select the **'black hole'** template.

3. The following path should be the SDDM themes directory:

```
/usr/share/sddm/themes/sddm-astronaut-theme/
```  

4. Replace the default configuration file:

```
Themes/black_hole.conf
```

with the one provided in this folder.  

5. Place the desired video in the `Backgrounds` folder.
